# HSC Python Practice App - How to Use

## Quick Start

1. **Open the app**: Simply double-click `index.html` or open it in any modern web browser (Chrome, Firefox, Edge, Safari)

2. **Wait for loading**: The first time you load the page, it will download the Python environment (Pyodide). This takes 10-30 seconds.

3. **Start practicing**: Click "Generate New Question" to get your first HSC-style Python question

## Features

### 📝 Detailed Questions
- 11 complete HSC-style questions (3-6 marks each)
- Questions match the format from requirements.md
- Full descriptions with examples and expected output
- Covers all HSC topics: loops, functions, classes, validation, data structures

### 💻 Code Editor
- Write Python code with syntax highlighting
- **Tab key inserts 4 spaces** (proper indentation)
- Starter code provided for each question

### ▶️ Run Code
- Click "Run Code" to execute your Python program
- See actual output from print() statements
- Error messages shown if code has issues

### ✓ Submit for Feedback
- Get encouraging, constructive feedback
- Checks if you're using appropriate concepts
- Identifies good practices in your code
- No harsh criticism - supportive for learners

### 🎲 Generate New Question
- Randomly selects from the question bank
- Practice different types of problems

## Hosting Options

This is a **pure HTML/CSS/JavaScript** application - no backend required!

### Local Use
- Just open index.html in a browser
- No server needed

### GitHub Pages
1. Create a GitHub repository
2. Push index.html to the repo
3. Enable GitHub Pages in Settings
4. Your app is live!

### Netlify
1. Drag and drop the folder to Netlify
2. Instant deployment

### Any Static Host
Works on: Vercel, Cloudflare Pages, AWS S3, Azure Static Web Apps, etc.

## Tips for Students

1. **Read the question carefully** - HSC questions have specific requirements
2. **Check the expected output** - Your program should match the example
3. **Use the Run button** - Test your code before submitting
4. **Tab key works** - Use it for proper indentation
5. **Don't worry about mistakes** - The feedback is supportive and helpful

## Browser Requirements

- Modern browser with JavaScript enabled
- Internet connection (first load only, to download Pyodide)
- After first load, works offline!

## Question Bank

The app includes 11 detailed questions covering:
- Diamond Grid Game (5 marks) - arrays, random, nested loops
- Game Scoring with Bonus (6 marks) - lists, conditionals, scoring logic
- Username Validation (3 marks) - string validation, functions
- Password Checker (3 marks) - string slicing, validation
- Student Grade Analysis (5 marks) - statistics, lists
- Phone Number Formatter (4 marks) - string manipulation
- Shopping Cart (6 marks) - dictionaries, calculations
- Email Validator (4 marks) - string validation
- Number Pyramid (4 marks) - nested loops, patterns
- Word Frequency Counter (5 marks) - dictionaries, text processing
- Student Class (6 marks) - OOP, classes, methods
